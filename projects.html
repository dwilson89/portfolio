Project: Silo Inspection UAV
Year: 2014
Role: Ground Control and Localisation
Languages Utilised: C#, C++, Arduino
Description: Develop a UAV system to successfully navigate in an indoor environment, and locate potential pests (Gypsie Moths, Straw).
Main tasks:

  • Document creation following a systems engineering approach, this including System Requirements, Trade studies, Preliminary/Final Design, and Test Reports.
  • Participation in group discussions including strategic planning, progress updates, and troubleshooting. This includes both with a client and without.
  • Research and technical analysis of current state system behaviour relating to my directed tasks.
  • Using C# to build upon an existing Ground Control Station software package to include additional user functionality for pest detection.
  • Developing with C++ in order to create an addition application to interface with other systems in use, such as (Vicon and Telecommunications subsystems). In addition to developing on Arduinos to act as an interface for the telecommunications between the Ground Control Station and the UAV in order to develop an indoor localisation system.
  • Planning and execution of test suites for user acceptance.

Key Achievements:

  • Received positive and consistent feedback from Team Lead on my approach to the project and the work I was producing
  • Received positive and consistent feedback from Course Coordinator, with documents produced used as a reference point for other students
  • Project team received a Distinction (6)
